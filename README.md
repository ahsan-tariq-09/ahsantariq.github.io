# Running Locally

Clone the repository:

```bash
git clone https://github.com/ahsan-tariq-09/ahsan-tariq-09.github.io.git
cd ahsan-tariq-09.github.io
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

The site will be available at:

```
http://localhost:5173
```

---

# Building the Site

Build the production version:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

---

# Deployment

The website is automatically deployed using **GitHub Actions** whenever changes are pushed to the `main` branch.

Deployment workflow:

1. Install dependencies
2. Build the Vite project
3. Upload build artifacts
4. Deploy to GitHub Pages

The deployed output is the `dist` directory generated by Vite.
